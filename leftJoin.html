LEFT JOIN Nümunəsi 🎓

Student ⟷ Mark (Tələbə ⟷ Qiymət)

SQL Sorğusu:

SELECT s.id, s.Name, m.Value
FROM Student s           ← SOL CƏDVƏL (əsas)
LEFT JOIN Mark m         ← SAĞ CƏDVƏL (əlavə)
ON s.Mark_Id = m.id

LEFT JOIN qaydası:

Student cədvəlinin BÜTÜN sətirlərini göstər.
Mark cədvəlində uyğunluq varsa - qiymətini göstər.
Uyğunluq yoxdursa - NULL göstər.

Vizual göstəriş (Venn Diaqramı)

🔵 Mavi = Sol cədvəl (Student) - hamısı qalır
🟢 Yaşıl = Ortaq hissə (uyğun olanlar)
⚪ Boz = Sağ cədvəl (istifadə olunmayan)

🔵 Sol cədvəl (Student)

FROM Student - bura SOL tərəfdir. Bütün sətirləri nəticədə görsənir.

🟢 Sağ cədvəl (Mark)

LEFT JOIN Mark - bura SAĞ tərəfdir. Yalnız uyğun gələnlər istifadə olunur.

⚠️ NULL nə vaxt olur?

Sol cədvəldə var, amma sağ cədvəldə uyğunluq yoxdursa NULL olur.
